## Full Description (for README header)

**Lumina Glass** is a full-stack personal knowledge workspace wrapped in a luminous dark-glass UI. Capture notes with markdown and bidirectional linking, reflect in a mood-tracked diary, and watch your ideas connect in an interactive knowledge graph — all inside a single, distraction-free sanctuary.

---

## Elevator Pitch (for social / portfolio)

Most note apps feel like filing cabinets. Lumina Glass feels like a room.  
It combines **linked notes**, **mood-aware journaling**, and a **living knowledge graph** into one dark, glassmorphic workspace — built with React, Hono, tRPC, and SQLite.
